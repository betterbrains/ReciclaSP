Recicla SP

O ReciclaSP é um projeto de código aberto para ajudar as pessoas encontrarem "Pontos de Entrega Voluntária (PEV)" para reciclagem de resíduos domésticos previamente separados.

O projeto pretende cobrir todo o estado de São Paulo, outros estados estão sendo avaliados para viabilizar outras versões específicas desse aplicativo.

Como essas informações não são disponibilizadas de forma oficial por nenhum orgão público, a maioria dos PEVs são encontrados via pesquisa na internet e informações de ONGs e empresas privadas, sendo assim o ReciclaSP será atualizado constantemente com novos PEV e novas funcionalidades.

Para colaborar com o projeto, pode-se criar "issues" propondo novas funcionalidades e melhorias, reportando bugs e claro, fazendo fork do projeto e enviando seus pull requests.
